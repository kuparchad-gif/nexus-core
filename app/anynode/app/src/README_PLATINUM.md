# Viren Platinum Edition

## Overview
Viren Platinum Edition is an enhanced version of the Viren AI Assistant with premium features including:

- Beautiful, modern interface with animations
- Document processing capabilities for PDF, DOCX, XLSX, and Visio files
- Model hot-swapping across different providers (Ollama, vLLM, LM Studio)
- Voice interaction with full-screen animation
- Design canvas for diagrams and flowcharts
- Secure authentication with MFA support
- Cloud connectivity for distributed monitoring

## Installation

### Prerequisites
- Python 3.8 or higher
- Ollama, vLLM, or LM Studio for model hosting
- Required Python packages (installed automatically by launcher)

### Quick Start
1. Run `run_viren_platinum.bat` to launch Viren Platinum Edition
2. The interface will open in your default web browser
3. Log in with the default credentials (admin/admin)
4. Explore the various tabs and features

## Features

### Chat Interface
- Modern, responsive chat interface
- Voice interaction with animated visualization
- Support for multiple models and providers
- Automatic model switching based on task

### Document Studio
- Upload and process various document types
- Extract text and metadata from documents
- Generate previews and visualizations
- Support for PDF, DOCX, XLSX, CSV, TXT, and Visio

### Design Canvas
- Create diagrams, flowcharts, and mockups
- Collaborate on designs in real-time
- Export designs in various formats
- Integration with document processing

### Model Lab
- Switch between different models on the fly
- Monitor model performance metrics
- Download new models automatically
- Compare model capabilities and performance

### System Monitoring
- Real-time system metrics
- Component health visualization
- Performance charts and logs
- Resource usage tracking

## Configuration
Configuration files are stored in the `config` directory:
- `viren_platinum_config.json`: Main configuration file
- `model_config.json`: Model definitions and settings
- `tts_config.json`: Text-to-speech settings
- `stt_config.json`: Speech-to-text settings

## Security
Viren Platinum Edition includes robust security features:
- User authentication with password hashing
- Multi-factor authentication (MFA) support
- Session management and timeout
- Role-based access control

## Cloud Connectivity
When deployed in the cloud, Viren Platinum Edition can:
- Monitor multiple instances across deployments
- Synchronize models and configurations
- Provide centralized management
- Enable distributed processing

## Troubleshooting
If you encounter issues:
1. Check the logs in the `logs` directory
2. Verify that all dependencies are installed
3. Ensure Ollama or your chosen model provider is running
4. Try restarting the application
