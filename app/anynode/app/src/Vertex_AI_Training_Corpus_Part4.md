### Google Cloud Platform (Vertex AI)

#### Core Services
- **Vertex AI**: Unified ML platform
  - AutoML: Automated machine learning
  - Custom Training: For specialized models
  - Model Garden: Pre-trained models
  - Feature Store: Feature management
  - Pipelines: ML workflow orchestration

- **BigQuery**: Serverless data warehouse
  - ML integration
  - Petabyte-scale analytics
  - Real-time analysis

- **Cloud Storage**: Object storage
  - Standard, Nearline, Coldline, and Archive storage classes
  - Global edge caching
  - Strong consistency

#### Free Tier Resources
- Vertex AI: Limited free credits for new users
- Cloud Functions: 2 million invocations per month
- Cloud Run: 2 million requests per month
- Cloud Storage: 5GB per month
- BigQuery: 10GB storage and 1TB queries per month

#### Best Practices
- Use preemptible VMs for batch processing
- Implement lifecycle policies for storage
- Leverage serverless options to minimize costs
- Use Cloud Scheduler for periodic tasks
- Implement proper IAM for security
